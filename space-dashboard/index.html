How This Analysis Was Conducted
Synthetic dataset of 3,000 space missions modeled on real-world program data, cleaned and analyzed entirely in SQL and Excel. No predictive model was trained — this is exploratory, descriptive analysis designed to surface actionable strategic insights.
01 · Data Source
Dataset & Scope

3,000 mission records spanning 10 spacefaring nations across 2000–2025. Each record captures mission outcome, budget, technology type, collaboration partners, environmental impact, and duration.

  • → 3,000 missions × 12 columns
  • → 10 countries, 2000–2025 (25 yrs)
  • → Binary success outcome per mission
  • → 5 propulsion technology categories
  • → Budget range: $0.58B – $49.97B
02 · Data Preparation
Cleaning & Engineering

Dataset validated for consistency across all 12 fields. Budget efficiency metric engineered as a derived column. Collaboration size extracted from partner-country lists. Environmental impact binned into three tiers.

  • → No missing values in source data
  • → Derived: success_rate / budget ($B)
  • → Partner count → collaboration tier
  • → YoY aggregation by calendar year
  • → Environmental impact: Low/Med/High
03 · Analysis
Findings & Techniques

Group-level aggregations computed across 5 analytical dimensions. SQL window functions (RANK, ROW_NUMBER) used to surface cost-effectiveness rankings. Cross-dimensional comparisons validated in Excel pivot tables.

  • → SQL · Excel · Data Visualization
  • → RANK() window fn for cost scoring
  • → GROUP BY across 5 dimensions
  • → Pivot cross-validation in Excel
  • → Descriptive analysis — no ML model
Key Analytical Decisions
Design Choices & Rationale
Efficiency Metric
Success rate divided by average budget in $B was chosen as the primary ROI proxy. This penalizes high-spend programs that don't proportionally improve outcomes and rewards lean execution.
YoY Trend Reading
25-year trend shows no statistically meaningful upward trajectory in success rate (range: 72.5%–77.1%). This was interpreted as a stable industry baseline, not stagnation — consistent with mature aerospace programs.
Collaboration Sizing
Partnership tiers (1–4 nations) were analyzed rather than specific bilateral pairs. This surfaces structural patterns — the 3-nation sweet spot — without overfitting to individual country dyads.
Cost-Effectiveness Score
Top-10 missions ranked using a composite score of success rate relative to budget. SQL RANK() window function applied over the full dataset to produce a normalized, reproducible ranking across all 3,000 records.
